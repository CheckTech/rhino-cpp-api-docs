Detailed Description

This class is a control which can be placed on a dialog to allow the user to set an angle.

◆ SetDialColors()

virtual void CRhRdkAngleCtrl::SetDialColors ( COLORREF  col1,
COLORREF  col2 
)
virtual

Set the colors of the circular dial when the style is kUnmarked, kCompass or kAzimuth. The dial will be shaded with a blend from col1 to col2 and the blend will rotate with the dial's angle setting.

Note
It is also possible to implement IRhRdkAngleDialDecorator to override these colors dynamically.
