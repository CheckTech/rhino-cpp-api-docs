bool CRhinoSdkRenderViewWnd::GetWireframeCompositeDIBs ( CRhinoDib &  dib1,
CRhinoDib &  dib2,

)

Operations.

Description: Create DIBs used to transparently blit wires to output render DIB. The DIB references passed to this function will contain bitmaps need when calling CRhinoSdkRenderViewWnd::DrawWiresInDIB( alpha_dib, wires_dib, final_dib). If alpha_dib or wires_dib contian an existing bitmap the existing bitmap will be destroyed and a new one created. This all dibs will be created using the specified render output size. Parameters: alpha_dib [out] - Alpha map for all wireframe and curves. wires_dib [out] - Rendered wireframes and curves in their display colors. bGetRenderSettingsFromDoc [in] - Tells compositor to setup render engine using the settings from the Document (true) or Display (false) settings. Returns: true If DIBs are created successfully otherwise returns false. Comments: See: CRhinoSdkRenderViewWnd::DrawWiresInDIB( CRhinoDib& alpha_dib, CRhinoDib& dib2, CRhinoDib& final_dib)
